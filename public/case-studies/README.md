# Case Study Slide Images

Place one PNG per slide from the **GVA Case Study Deck (1)** here:

- `slide1.png` … `slide16.png`

These are used in:

1. **Benefit modals** — Success Stories tab shows the slide image for that benefit.
2. **Generated Value Deck (PPTX)** — A "Case Studies" section (before Appendix) includes one slide per image.

## How to create the images

1. Open **GVA Case Study Deck (1).pptx**.
2. Export each slide as PNG (e.g. File → Export → Images, or copy slide as image and save).
3. Name them `slide1.png`, `slide2.png`, … `slide16.png` (slide 1 = first slide in the deck).
4. Put all files in this folder (`public/case-studies/`).

If a file is missing, the Success Stories tab for that benefit will still work when the image exists; the generated deck will skip that case study slide if the image fails to load.
