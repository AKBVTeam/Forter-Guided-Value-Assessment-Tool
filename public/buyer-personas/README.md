# Buyer Persona PDFs

Copy these 6 PDF files into this folder so the "Buyer Personas" tab in the GVA ? overview can offer downloads:

- **Payments Director Persona Card.pdf**
- **Fraud Director Persona Card.pdf**
- **Digital - e-Commerce Director Persona Card.pdf**
- **Chief Financial Officer Persona Card.pdf**
- **Chief Technology Officer Persona Card.pdf**
- **Chief Digital Officer Persona Card.pdf**

Use the **exact filenames** above (including spaces and hyphens). The app links to them at `/buyer-personas/<filename>`.
